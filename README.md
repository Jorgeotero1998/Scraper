Desafio de Scraping - TypeScript. Este proyecto implementa un scraper robusto en TypeScript para la extraccion de datos y descarga de documentos sin usar navegadores automatizados. Caracteristicas principales: Estrategia HTTP pura utilizando axios y cheerio, resiliencia ante errores 429 con Backoff Exponencial integrado, y persistencia de datos en formatos JSON, CSV y failed_downloads.json. Comandos de ejecucion: Para OEFA usar npm run start:oefa y para Poder Judicial usar npm run start:pj
